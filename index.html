Notes: Memory map: All 64K ($0000-$ffff) is populated with RAM and available. Page $00: 6502 zero page, used for variables and pointers Two locations in the zero page are connected to devices: * Memory location $fe contains a new random byte on every instruction. * Memory location $ff contains the ascii code of the last key pressed. Values written to $ff will remain unchanged until next key press. Printable keys have their ASCII value; BackSpace is $08; Enter is $0d; Cursor controls are $80=up, $81=right, $82=down, $83=left Page $01: 6502 stack (downward) Pages $02-05: bitmapped display Pages $06-$ef: your code and data Page $f0-$f7: character display Page $f8-$ff: reserved for future ROM emulation/system routines Available directives: dcb value[,value[...]] - (define constant byte) used to create constants in memory. Multiple bytes are separated by commas. Values may be in decimal, hexadecimal, or single characters in double quotes (except spaces - use 32 or $20 to represent a space). define macro value - defines a macro expression. value must be an opaque string. *=address - changes assembler pointer to given address label: - defines an address label. Reference the label in your code without the colon. Labels can be used with the "Jump To" button in the debugger. Displays: The bitmapped display is 32x32 pixels, and memory locations $200 to $5ff map to the screen pixels. Different values will draw different colour pixels. The colours are: $0: Black $1: White $2: Red $3: Cyan $4: Purple $5: Green $6: Blue $7: Yellow $8: Orange $9: Brown $a: Light red $b: Dark grey $c: Grey $d: Light green $e: Light blue $f: Light grey The character display is 80x25 characters in size, and memory locations $f000-$f7cf contains the ASCII characters on the display (row-major). Bit 7, if set, make a character appear in reverse video. Codes $21-$7e (and corresponding codes with bit 7 set, $a0-fe) display; all other codes show up as blanks. Reference: A reference for the 6502 opcodes is available at http://www.6502.org/tutorials/6502opcodes.html
