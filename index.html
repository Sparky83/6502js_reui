Notes: Memory location $fe contains a new random byte on every instruction. Memory location $ff contains the ascii code of the last key pressed. The bitmapped display is 32x32 pixels, and memory locations $200 to $5ff map to the screen pixels. Different values will draw different colour pixels. The colours are: $0: Black $1: White $2: Red $3: Cyan $4: Purple $5: Green $6: Blue $7: Yellow $8: Orange $9: Brown $a: Light red $b: Dark grey $c: Grey $d: Light green $e: Light blue $f: Light grey The character display is 80x25 characters in size, and memory locations $f000-$f7cf contains the ASCII characters displayed in this region. The DCB (define constant byte) directive accepts decimal, hex, or single ASCII characters quoted with double quotes. However, literal spaces must be encoded in decimal (32) or hex ($20). A reference for the 6502 opcodes is available at http://www.6502.org/tutorials/6502opcodes.html
